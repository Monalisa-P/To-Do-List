To run this project, follow these methods.
Method 1
Step 1: 
  Open your "Terminal".
Step 2: Clone your repository locally
  Type "git clone https://github.com/Monalisa-P/To-Do-List.git"
Step 3: Navigate to the Directory
  Type "cd To-Do-List"
Step 4: Run the script
  Type "python main.py"
